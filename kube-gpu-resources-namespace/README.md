# Kube GPU Resource Namespace

Grafana dashboard for GPU resource by namespace.

This dashboard provide following things by namespace.

- Summary
  - Status
    - quota
    - reserved
    - usage
    - util
  - Trends
    - quota
    - reserved
- by Pod
  - reserved
  - usage
  - util
- by Container
  - reserved
  - usage
  - util

## Install

Import [data.json](./data.json) to your Grafana dashboard.
